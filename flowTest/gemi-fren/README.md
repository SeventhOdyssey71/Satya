# Gemi-Fren: Advanced Agent Knowledge Base

This folder contains comprehensive knowledge files to enhance the Gemini AI agent's performance on the Satya platform. These files provide detailed context, examples, and best practices for optimal task execution.

## Knowledge Files Overview

- `platform-architecture.md` - Complete technical architecture and implementation details
- `agent-capabilities.md` - Detailed agent capabilities and action frameworks
- `user-interaction-patterns.md` - User intent patterns and response strategies
- `data-visualization-specs.md` - Data rendering and visualization guidelines
- `api-integration-guide.md` - Service integration patterns and error handling
- `response-optimization.md` - Response quality and optimization techniques
- `task-execution-flows.md` - Workflow patterns for complex tasks
- `troubleshooting-guide.md` - Common issues and resolution strategies
- `platform-examples.md` - Real-world usage examples and case studies
- `advanced-prompting.md` - Advanced prompting techniques for complex queries

## Usage

These files are automatically loaded into the agent's context when processing user requests, enabling sophisticated understanding and execution of platform-specific tasks.
